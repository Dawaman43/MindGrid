This roadmap outlines the upcoming milestones for MindGrid. Update this page
when timelines or owners change.

## Current milestone (Q1)

- **Focus:** Onboarding, workspace creation, and core grid interactions.
- **Primary owners:** Product + Web.
- **Dependencies:** Auth/session stability, analytics baseline.

## Next milestone (Q2)

- **Focus:** Collaboration features and activity streams.
- **Primary owners:** Platform + Web + Mobile.
- **Dependencies:** Real-time sync, notification infrastructure.

## Shared dependencies

- **Design system refresh:** Align tokens and component usage.
- **API surface expansion:** Workspace and grid enhancements.
- **Observability:** Add tracing for grid creation and sharing.

## Decision log

- **2024-11-05:** Align on typed RPC for client-server communication.
- **2024-11-12:** Prioritize onboarding flow and grid templates.
- **2024-11-19:** Commit to unified release readiness checklist.

## Ownership

- **Roadmap driver:** Product lead.
- **Platform lead:** Owns infrastructure sequencing.
- **Client leads:** Web and mobile implementation.
